# 📖 Pokedex

Una aplicación web desarrollada en **Vue.js** que permite consultar información de Pokémon utilizando la [PokeAPI](https://pokeapi.co/).  
Este proyecto fue creado con fines educativos y prácticos, para aprender sobre consumo de APIs, manejo de estados, componentes y diseño de interfaces modernas con Vue.

---

## ✨ Características principales

- 📋 **Listado completo de Pokémon** con sus imágenes oficiales.
- 🔍 **Búsqueda dinámica** por nombre o número de Pokémon.
- 📊 **Visualización de estadísticas**: tipos, habilidades, peso, altura, etc.
- 🎨 **Interfaz responsiva y amigable**, adaptable a dispositivos móviles.
- ⚡ **Consumo en tiempo real** de datos desde la PokeAPI.
- 🧩 Código modular y organizado para facilitar la comprensión y mantenimiento.

---

## 🛠️ Tecnologías utilizadas

- [Vue.js](https://vuejs.org/)  
- [JavaScript](https://developer.mozilla.org/es/docs/Web/JavaScript)  
- [CSS](https://developer.mozilla.org/es/docs/Web/CSS)  
- [PokeAPI](https://pokeapi.co/)

## 📦 Instalación y uso

Sigue estos pasos para ejecutar el proyecto en tu máquina local:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/JuanSposada/Pokedex.git
   ```
2. Accede a la carpeta del proyecto: `cd Pokedex`
3. Instala las dependencias: `npm install`
4. Inicia la aplicación: `npm run serve`

# 📂 Estructura del proyecto

```text
Pokedex/
├── public/              # Archivos públicos y estáticos
├── src/
│   ├── assets/          # Recursos como imágenes y estilos
│   ├── components/      # Componentes reutilizables de Vue
│   ├── views/           # Vistas principales de la aplicación
│   ├── services/        # Lógica para consumir la PokeAPI
│   ├── router/          # Configuración de rutas
│   ├── store/           # Manejo de estado con Vuex (si aplica)
│   ├── App.vue          # Componente raíz
│   └── main.js          # Punto de entrada de la aplicación
├── package.json         # Dependencias y scripts
└── README.md            # Documentación del proyecto
```
